Erm&ouml;glicht das automatisierte Umschreiben von deutschen Umlauten in URLs in einen lesbaren Permalink.

== Description ==

Erm&ouml;glicht das automatisierte Umschreiben von deutschen Umlauten und Sonderzeichen in URLs f&uuml;r Artikel, Seiten, Costum Post Types, Kategorien, Schlagw&ouml;rter und Dateien in einen lesbaren Permalink.

Zus&auml;tzliche Funktionen:

* Alte Permalinks k&ouml;nnen nachtr&auml;glich umgeschrieben werden (automatisiert/optional)
* Unterst&uuml;tzung von Costum Post Types
* Unterst&uuml;tzung von Mediathek/Dateinamen
* Stopwords: Entfernt automatisiert gew&ouml;hnliche W&ouml;rter (der, die, das, ein, eine usw.) aus dem Permalink (optional)
* Unterst&uuml;tzung aller Buchstaben der Zeichenkodierung ISO 8859-1 (Latin-1)

== Installation ==

1. Entpacke `permalauts-plus.1.0.zip`.
2. Lade den `permalauts-plus`-Ordner in dein Verzeichnis `/wp-content/plugins` hoch.
3. Aktiviere das Plugin im 'Plugins'-Men&uuml; der Wordpress-Administration.
4. Konfiguriere das Plugin unter Permalauts Plus.
